# 📝 TaskFlow

TaskFlow is a modern and responsive To-Do application built using **HTML5** and **CSS3**. Inspired by Microsoft To Do and Apple Reminders, it features a clean, minimal interface designed to provide an intuitive task management experience.

## ✨ Features

* Modern and responsive user interface
* Search bar UI
* Task statistics cards
* Add Task form
* Category chips
* Today's Tasks section
* Completed Tasks section
* Smooth hover animations
* Semantic HTML5 structure
* Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Fraunces & Manrope)
* Font Awesome
